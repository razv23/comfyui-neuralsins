# Moved to Symbiotica

This repository has been merged into **Symbiotica** — a single, unified ComfyUI pack that contains everything Neural Sins did plus the Symbiotica agent system.

## Where to find the nodes now

- **Repo:** https://github.com/symbiotica-ai/comfyui-nodes
- **ComfyUI Registry:** https://registry.comfy.org/nodes/symbiotica
- **ComfyUI Manager:** search **Symbiotica**

## Existing workflows

All node class names were preserved (still `NS*`-prefixed), so workflows saved with Neural Sins continue to load after switching to the Symbiotica pack — just uninstall `comfyui-neuralsins` and install `symbiotica`.

## Why

Maintaining two packs in parallel was overhead with no upside. One install for everything is the goal.

This repository is archived. No further updates will be pushed here.
